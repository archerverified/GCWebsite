# Open Graph Images

This directory should contain Open Graph images for social media sharing.

## Required Images

1. **og-default.png** (1200x630px)
   - Default Open Graph image for all pages
   - Should include:
     - Garage Cowboy logo/branding
     - "24/7 Professional Garage Door Service" tagline
     - Phone number: (817) 256-0122
     - "Dallas-Fort Worth Metroplex" text
     - Yellow (#FEC300) and dark gray (#35363A) brand colors

## Image Guidelines

- **Dimensions**: 1200x630 pixels (recommended by Facebook/Twitter)
- **File Size**: Keep under 8MB (ideally under 1MB)
- **Format**: PNG or JPG
- **Text**: Ensure text is readable on mobile devices

## Creating Images

You can create Open Graph images using:
- Canva (free templates available)
- Figma (design tool)
- Adobe Photoshop/Illustrator
- Online OG image generators

## Testing

Test your Open Graph images using:
- [Facebook Sharing Debugger](https://developers.facebook.com/tools/debug/)
- [Twitter Card Validator](https://cards-dev.twitter.com/validator)
- [LinkedIn Post Inspector](https://www.linkedin.com/post-inspector/)
